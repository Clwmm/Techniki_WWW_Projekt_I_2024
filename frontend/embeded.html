Embedded Projects

Raspberry Pi Pico Maze Generator

This repository contains a project for procedural creating and solving a maze using a Raspberry Pi Pico microcontroller and a Pico-ResTouch-LCD-3.5 display. The maze solving algorithm is implemented using: Maze generation algorithm and A* search algorithm.

Arduino Nano Clock

The Arduino Nano Clock repository is a project aimed at creating a versatile and customizable clock using the Arduino Nano microcontroller board. This project provides a platform for hobbyists, makers, and enthusiasts to build their own digital clocks with various features and functionalities.
